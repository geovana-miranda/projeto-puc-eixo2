# Programação de Funcionalidades

Implementação do sistema descrita por meio dos requisitos funcionais e/ou não funcionais. Nesta seção, é essencial relacionar os requisitos atendidos com os artefatos criados (código fonte) e com o(s) responsável(is) pelo desenvolvimento de cada artefato a cada etapa. Nesta seção também deverão ser apresentadas, se necessário, as instruções para acesso e verificação da **implementação que deve estar funcional no ambiente de hospedagem obrigatoriamente a partir da Etapa 03**.

|ID    | Descrição do Requisito  | Artefatos produzidos | Aluno(a) responsável |
|------|-----------------------------------------|----|----|
|RF-001| O sistema deve permitir que o usuário faça cadastro e login| FormularioCriarConta.jsx, FormularioLogar.jsx, Formulario.module.css | Geovana |
|RF-002| O sistema deve permitir a criação e exclusão de quadros| ModalCriarQuadro.jsx, ModalAlterarQuadro.jsx ModalCriarQuadro.module.css | Thiago |
|RF-003| O sistema deve permitir que o usuário  pesquise por um quadro | Pesquisa.jsx | Thiago |
|RF-004| O sistema deve permitir adicionar membros aos quadros | ModalCriarQuadro.jsx, ModalAlterarQuadro.jsx ModalCriarQuadro.module.css | Geovana |
|RF-005| O sistema deve permitir que o administrador remova membros do quadro | ModalCriarQuadro.jsx, ModalAlterarQuadro.jsx ModalCriarQuadro.module.css | Geovana |
|RF-007|O sistema deve permitir que o administrador gerencie o quadro | ModalCriarQuadro.jsx, ModalAlterarQuadro.jsx ModalCriarQuadro.module.css | Thiago |
|RF-008| O sistema deve permitir a criação e exclusão de listas dentro de cada quadro | CriarLista.jsx, CriarLista.module.css, Lista.jsx, Lista.module.css | Geovana e Thiago |
|RF-009| O sistema deve permitir a criação e exclusão de tarefas dentro de cada lista | Tarefa.jsx, Tarefa.module.css | Geovana |
|RF-010| O sistema deve permitir adicionar uma descrição da tarefa | ModalAlterarTarefa.jsx, ModalAlterarTarefa.module.css  | Geovana |
|RF-011| O sistema deve permitir adicionar a cada tarefa uma data de início e uma data de entrega | ModalPropriedades.jsx, ModalPropriedades.module.css | Thiago |
|RF-012| O sistema deve permitir atribuir uma tarefa a um membro | ModalAtribuirTarefa.jsx, ModalAtribuirTarefa.module.css | Geovana |
|RF-011| O sistema deve permitir que o usuário classifique a prioridade de uma tarefa em baixa, média e alta | ModalPropriedades.jsx, ModalPropriedades.module.css | Thiago |
|RF-012| O sistema deve permitir mover um item da tarefa para outra lista  | Não implementado |
